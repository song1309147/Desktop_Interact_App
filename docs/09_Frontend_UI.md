# 桌面互动软件 - 09_前端设计文档 (UI/UX 与视觉载体)

## 1. 模块概述
本模块负责构建系统的“物理外壳”与“控制中枢”。它涵盖了桌面端无边框透明窗口的绘制、AI 动态生成的 Q 版角色动画逻辑、文字气泡的渲染，以及供用户调节底层参数的设置面板。本模块的设计核心是**“极低侵入性的视觉陪伴”**与**“高度个性化的形象定制”**。

## 2. 核心视觉：AI 动态形象与动画逻辑 (Dynamic Avatar)
为控制开发成本并实现“千人千面”的专属羁绊，前端角色表现采用 **AI 动态生成的 Q 版帧序列（PNGTuber 模式）**。

### 2.1 形象生成与兜底机制
* **首次启动引导**：软件初次运行会弹出向导，用户可输入期望的角色特征（如：银发、机甲风、Q版小男孩），系统根据 04 模块的性格设定，调用 AI 绘画接口动态生成一组专属的去背建立绘。
* **基准表情库 (Expression Library)**：AI 需固定生成以下 5 种状态的差分图，以匹配 08 模块的状态机：
  1. `Neutral/Listening` (默认/侧耳倾听)
  2. `Speaking` (张嘴说话)
  3. `Thinking` (闭眼/圈圈眼，表示处理中)
  4. `Happy` (赞赏/开心，针对游戏高光时刻)
  5. `Shocked/Sad` (惊讶/安慰，针对报错或危险状态)
* **默认兜底**：若用户跳过捏脸流程，系统自动加载内置的“温柔青年女性”Q 版基准图组。

### 2.2 基础动画逻辑 (Animation Engine)
* **呼吸感模拟**：前端通过 CSS `transform` 或图形库矩阵变换，对当前静态图进行周期性的微量上下浮动与纵向拉伸（周期约 3 秒），模拟角色呼吸。
* **唇音同步 (Audio-Visual Sync)**：处于 `SPEAKING` 状态时，前端监听 05 模块的音频输出音量。当音量超过阈值时切换为“张嘴”图，低于阈值时切换为“闭嘴（默认）”图，实现轻量级的说话动画。

## 3. 桌面 UI 交互与防误触 (UI Interaction)
挂件必须在桌面上做到“隐形”的陪伴，交互需依赖特定的修饰键：

* **无边框透明窗口**：系统不渲染常规的 Windows/Mac 窗口边框，背景完全透明。
* **全局鼠标穿透**：默认状态下，鼠标点击直接穿透挂件，作用于后方游戏或代码编辑器。
* **拖拽与吸附 (Drag & Snap)**：按住 `Alt` 键 + 鼠标左键，挂件实体化并可随意拖动。靠近屏幕边缘时触发“果冻弹性”吸附。
* **右键快捷菜单**：`Alt` + 鼠标右键唤出极简环形菜单，包含：[打开设置]、[休眠挂件]、[强制闭嘴]。
* **气泡动态渲染**：文字采用“打字机”效果逐字浮现，气泡背景采用亚克力（毛玻璃）模糊材质，确保文字清晰且不完全遮挡游戏画面。

## 4. 设置面板设计 (Settings Console)
设置面板是软件的后台控制台，分为以下四个核心区块：

### 4.1 核心驱动 (API & 模型)
* **API Key 凭证**：密码框显示，支持一键测试连通性，修改后触发热重载（无需重启）。
* **模型路由切换**：下拉选择主力云端大模型（如 Gemini）或本地兜底模型。

### 4.2 灵魂与形象 (Persona)
* **人设参数盘**：编辑 04 模块的性格标签与用户身份设定。
* **形象重塑 (Re-roll Avatar)**：随时丢弃当前 Q 版形象，重新呼出 AI 生成向导。
* **视觉缩放**：滑块控制挂件在桌面上的全局缩放比例。

### 4.3 行为与自动化 (Behavior & Automation)
* **抓取频率控制 (Capture Interval)**：滑块组件。允许用户自定义 02 模块引擎 B 的**“最大静默抓取间隔”**（如 3分钟、5分钟、10分钟）。滑块左侧硬编码限制**不得低于 15 秒**。
* **智能避让开关**：开启后，文字气泡将自动计算并反向弹出，躲避屏幕边缘及鼠标当前位置。

### 4.4 感官控制 (Audio & Input)
* **发声控制**：TTS 语音输出的音量大小滑块。
* **麦克风灵敏度 (VAD Threshold)**：提供一个实时跳动的音频音量条，用户可拖动基准线，过滤环境键盘声杂音。
