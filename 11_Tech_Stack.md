# 桌面互动软件 - 11_技术栈文档 (Tech Stack Blueprint)

## 1. 模块概述
本文档定义了“赛博好友”桌面互动软件的底层技术选型。本方案专为 **“AI 辅助编程 (AI-Assisted Coding)”** 打造，整体架构采用 **Python 后端 + Web 前端融合** 的模式。旨在利用 Python 强大的系统级接口处理视觉与 AI 逻辑，同时借助 HTML/CSS 实现丝滑的 Q 版角色动画与气泡特效，确保软件在运行 3A 大作或进行重度开发时，依然保持极低的资源占用。

## 2. 前端展示层 (Frontend UI & Animations)
为实现“透明无边框、果冻吸附、平滑呼吸感”等高级视觉反馈，彻底抛弃死板的传统 GUI 框架，采用网页技术渲染桌面挂件。
* **核心渲染容器**：`PyWebView` 或基于 PySide6 的 `QWebEngineView`。它能在桌面上抠出一个绝对透明的网页窗口，并支持鼠标穿透（Click-Through）控制。
* **布局与动画引擎**：原生 `HTML5` + `CSS3` + `JavaScript`。
  * 依托 CSS3 的 `transform` 和 `transition` 实现 Q 版形象的呼吸浮动、张嘴/闭嘴的差分图无缝切换。
  * 依托 CSS 的 `backdrop-filter: blur()` 实现文字气泡的亚克力（毛玻璃）质感。

## 3. 后端逻辑与视觉引擎 (Backend & Vision Core)
全业务逻辑（状态机控制、记忆读写、截图打码）由 Python 统揽。
* **开发语言**：`Python 3.10+` （兼顾新特性与第三方库的极佳兼容性）。
* **极速截屏引擎**：`mss`。
  * 舍弃老旧的 PIL/ImageGrab，采用纯 C 语言底层的 `mss`。实测在硬核动作游戏高帧率运行期间，仍能以毫秒级速度完成无感截图，完全不抢占游戏渲染流水线。
* **异步网络通信**：`httpx`。
  * 负责与云端大模型（如 Gemini）或本地大模型后台通信，支持全异步（Asyncio）并发，确保等待 AI 思考期间主线程绝对不卡顿。

## 4. 听觉交互模块 (Audio & Voice System)
本模块兼顾“零成本、高隐私、极速响应”，音频处理流全部本地化闭环。
* **人声端点检测 (VAD)**：`webrtcvad`。Google 开源的工业级声音检测库，精准剥离机械键盘敲击声，仅在捕捉到人类声纹时激活录音。
* **语音转文字 (STT / ASR)**：本地部署 `FunASR` 或 `Whisper-Tiny`。充分压榨现代 GPU 显存与统一内存算力，声音数据不出本地，实现零延迟的中英文混合识别，极大保障隐私。
* **文字转语音 (TTS)**：`edge-tts`。逆向调用微软 Edge 浏览器的免费高品质语音合成接口，免除 API 额度焦虑，自带多种高亲和力的自然女声。

## 5. 记忆与数据持久化 (Memory & Storage)
无需安装庞大的数据库软件，采用 Python 原生轻量级方案：
* **短期情景记忆**：内置 `SQLite3`。完美支撑“7天/1000条”双阈值滚动清理逻辑，提供极快的高频聊天数据检索能力。
* **长期灵魂记忆与配置**：原生 `json` 模块。将用户的性格档案、API 密钥、挂件刷新频率等数据序列化存储为本地 `.json` 或 `.env` 配置文件，支持热重载。
